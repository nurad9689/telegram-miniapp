# Настройка Telegram Mini App

## Что было реализовано

### 1. Раздел "Мероприятия" ✅
- Страница с отображением всех мероприятий
- Кнопка для просмотра на карте (заглушка)

### 2. Раздел "Локации" ✅
- Страница с отображением всех локаций
- Показывает адрес, рейтинг и описание
- Кнопка для просмотра на карте (заглушка)

### 3. Раздел "Участники" ✅
- Страница с отображением всех пользователей
- Показывает имя и рейтинг

### 4. Форма создания нового мероприятия ✅
- Полная форма с настройками:
  - Дата и время
  - Выбор локации
  - Формат игры (количество команд, участников)
  - Правила (условие окончания, ротация)
- Интеграция с Telegram MainButton

### 5. Форма создания нового мероприятия ✅
- Полная форма с настройками:
  - Адрес
  - Рейтинг (по умолчанию 0.0)
  - Описание
- Интеграция с Telegram MainButton

### 6. Интеграция с Telegram WebApp API ✅
- Создан хук `useTelegram` для работы с Telegram API
- Получение данных пользователя (ID, имя, username)
- Использование Telegram MainButton для создания мероприятий
- Показ нативных алертов
- Кнопка "Назад" в Telegram

## Как подключить приложение к Telegram

### Шаг 1: Развертывание фронтенда

Вам нужно развернуть фронтенд на HTTPS-сервере (Telegram требует HTTPS). Варианты:

**Вариант A: Vercel (рекомендуется)**
```bash
cd frontend
npm run build
# Загрузите папку build на Vercel
```

**Вариант B: Netlify**
```bash
cd frontend
npm run build
# Загрузите папку build на Netlify
```

**Вариант C: Любой хостинг с HTTPS**
- Разместите собранный проект на любом хостинге с поддержкой HTTPS

### Шаг 2: Создание бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните токен бота

### Шаг 3: Настройка Mini App

1. В @BotFather отправьте `/newapp`
2. Выберите вашего бота
3. Введите название приложения
4. Введите URL вашего развернутого фронтенда (например: `https://telegram-miniapp-eosin-theta.vercel.app`)
5. BotFather выдаст вам URL для Mini App

### Шаг 4: Тестирование

1. Откройте полученный URL в браузере
2. Приложение должно работать в режиме веб-приложения
3. Для тестирования в Telegram:
   - Откройте вашего бота
   - Нажмите на меню (или кнопку, если настроено)
   - Откройте Mini App

## Доступные функции

### API Эндпоинты

**Локации:**
- `POST /locations/` - Создать локацию
- `GET /locations/` - Получить все локации
- `GET /locations/{location_id}` - Получить локацию по id
- `PUT /locations/{location_id}` - Обновить локацию по id
- `DELETE /locations/{location_id}` - Удалить локацию по id

**Участники:**
- `POST /participants/` - Создать участника
- `GET /participants/` - Получить всех участников
- `GET /participants/{participant_id}` - Получить участника по id
- `PUT /participants/{participant_id}` - Обновить участника по id
- `DELETE /participants/{participant_id}` - Удалить участника по id

**Мероприятия:**
- `POST /events/` - Создать мероприятие
- `GET /events/` - Получить все мероприятия
- `GET /events/{event_id}` - Получить мероприятие по id
- `PUT /events/{event_id}` - Обновить участника по id
- `DELETE /events/{event_id}` - Удалить мероприятие по id

### Страницы приложения

- `/` - Мероприятия (главная страница)
- `/locations` - Локации
- `/rating` - Рейтинг игроков
- `/create-event` - Создание мероприятия
- `/create-location` - Создание мероприятия
